มารู้จักธนัศฌ์กัน!!!



ธนัศฌ์ อภิชนพงศกร



ธนัศฌ์ อภิชนพงศกร เป็นคนราชบุรี เกิดปลายทศวรรษที่ 90s จบการศึกษาชั้นมัธยมต้นจากโรงเรียนเบญจมราชูทิศ ราชบุรี จบการศึกษาชั้นมัธยมปลายจากโรงเรียนโยธินบูรณะ และเข้าศึกษาต่อที่คณะรัฐศาสตร์ สาขาความสัมพันธ์ระหว่างประเทศ ที่จุฬาลงกรณ์มหาวิทยาลัย หลังเรียนจบได้เข้าทำงานเป็น Professional Conference Organizer เป็นเวลา 2 ปีก่อนที่จะกลับมาทำงานร้านเครื่องเขียนของครอบครัวที่จังหวัดราชบุรีถึงปัจจุบัน

ปัจจุบัน มีความสนใจเกี่ยวกับเทคโนโลยี iot, การเขียน code ต้องการที่จะเปิด Code Dojo เพื่อให้เด็กๆ ในจังหวัดราชบุรีได้มีแหล่งเรียนรู้เกี่ยวกับเทคโนโลยีใหม่ นอกจากนี้ยังหันมาใส่ใจเกี่ยวกับด้านสุขภาพมากขึ้น หากมีเวลาว่างจะฝึกโยคะ คาโปเอร่า Free Running หรือกีฬากลุ่ม Movement อิ่นๆ
